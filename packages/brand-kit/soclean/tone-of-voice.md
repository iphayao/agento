# SoClean — Tone of Voice

## Core Personality

SoClean speaks like a **knowledgeable friend who genuinely cares about your daily comfort** — not a corporate brand, not a hard-sell promoter.

### Four Personality Traits

| Trait | What it sounds like | What it avoids |
|---|---|---|
| **Warm** | "เหมือนกำมือนุ่มๆ ทุกครั้งที่หยิบใช้" | Cold, clinical, distant |
| **Honest** | "ฝุ่นน้อย ไม่ได้บอกว่าไม่มีเลย" | Exaggerated absolute claims |
| **Practical** | "เหมาะสำหรับบ้าน ออฟฟิศ หรือร้านค้า" | Aspirational lifestyle fluff |
| **Confident** | "กระดาษทิชชู่ที่คุณเลือกซ้ำ" | Pushy, desperate, salesy |

## Thai Copywriting Rules

### Language Register
- Use **ภาษาไทยกลาง** (standard Thai) as the base register.
- For TikTok and Facebook organic: shift slightly toward **ภาษาพูด** (conversational Thai) to feel native.
- For Shopee/Lazada product listings: keep **formal-practical** Thai.
- Avoid slang that ages quickly or excludes older buyers.

### Sentence Style
- Short, punchy sentences for headlines.
- Conversational rhythm — read it aloud; it should sound natural.
- Never start a headline with a prohibited claim (see `prohibited-claims.md`).
- Use question hooks sparingly but effectively: "เคยรำคาญฝุ่นทิชชู่ไหม?"

### Punctuation and Emoji
- Emoji: use sparingly on TikTok and Facebook; avoid on Lazada/Shopee titles.
- Exclamation marks: maximum 1 per short copy block. More reads as desperation.
- Ellipsis (…): acceptable for dramatic pause in social copy.

## English Copywriting Rules

English copy is secondary and used primarily for:
- Export-facing or dual-language Shopee/Lazada listings.
- Internal brand documentation.

English tone mirrors the Thai: clear, friendly, no fluff.

### English Style
- Plain English. No marketing buzzwords.
- "Soft, low-dust tissue for everyday use." — not "The Ultimate Softness Experience."
- Active voice preferred.
- British or American English both acceptable — pick one and be consistent per asset.

## Voice Spectrum

Use this spectrum to calibrate tone per channel and content type.

```
Playful / TikTok video hooks
  |
  | ← Use for: TikTok entertainment content, Facebook stories
  |
Warm / Social product posts
  |
  | ← Use for: Facebook product posts, TikTok demo content
  |
Practical / Informational
  |
  | ← Use for: Shopee/Lazada descriptions, FAQ content
  |
Neutral / Factual
  |
  | ← Use for: compliance disclosures, reseller materials
```

## Do / Don't Examples

### Headlines

| Do | Don't |
|---|---|
| "เนียนนุ่ม ฝุ่นน้อย เหมาะทุกวัน" | "ไร้ฝุ่น 100% ปลอดภัยที่สุด!" |
| "ทิชชู่ที่บ้านเลือกซ้ำทุกเดือน" | "ทิชชู่อันดับ 1 ของไทย" (unverified) |
| "ซื้อยกลัง คุ้มกว่าเป็นไหนๆ" | "ถูกที่สุด! ลดราคาสูงสุด!" |
| "สัมผัสความนุ่มที่ต่างออกไป" | "นุ่มที่สุดในตลาด" |

### Body Copy

| Do | Don't |
|---|---|
| "ทิชชู่ 2 ชั้น 180 แผ่น ฝุ่นน้อย สัมผัสนุ่ม" | "กระดาษทิชชู่ medical-grade ปลอดภัยทางการแพทย์" |
| "เหมาะสำหรับหน้า มือ หรือเช็ดโต๊ะ" | "ฆ่าเชื้อโรค antibacterial" |
| "ซื้อยกลัง 50 แพ็ค ราคาส่งสำหรับร้านค้า" | "hypoallergenic เหมาะผู้แพ้ง่าย" (ไม่มีหลักฐาน) |

## Content Length Guidelines

| Format | Recommended length |
|---|---|
| TikTok caption | 50–100 characters Thai |
| Facebook post | 100–200 characters Thai + image |
| Shopee product title | 100 characters max (platform limit) |
| Shopee description | 500–1,000 characters Thai |
| Lazada product title | 120 characters max |
| Lazada description | 500–800 characters Thai |
